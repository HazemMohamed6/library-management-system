# Library Management System 📚

## 📌 Description
A console-based library management system built using Java, OOP, JDBC, and MySQL.  
This project implements core features of a library system including searching, borrowing, and transaction handling.

---

## 🚀 Features Implemented
- Add Books to the library
- Borrow Books with Transaction management
- Search Books by title
- Pagination for search results

> Note: Console UI/menu system has not been implemented yet. Operations are tested via `Main.java` calls.

---

## 🛠️ Technologies Used
- Java
- OOP (Inheritance, Abstract Classes)
- JDBC
- MySQL
- SOLID Principles
- Transactions
